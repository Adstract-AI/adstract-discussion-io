# Privacy Policy

Last Updated: March 5, 2026

This Privacy Policy explains how the **Adstract Discussion Assistant** Chrome extension ("Extension", "Service") handles data.

By using the Extension, you acknowledge this Privacy Policy.

---

## 1. Scope

This policy applies to data processed by the Extension while you use supported BigBlueButton pages.

---

## 2. Data We Process

The Extension may process:

- visible chat messages in the current session
- metadata related to those messages (for example: username, timestamp, student index when present)
- your generated/edited participation drafts
- local session/discussion history you create in the Extension
- configuration values you set (for example model, language, context size, prompt semantics)
- OpenAI API key you explicitly provide

---

## 3. Where Data Is Stored

The Extension stores session data and settings in browser extension storage (`chrome.storage.local` and `chrome.storage.session`).

Data is stored locally in your browser profile unless sent to external AI services.

---

## 4. External AI Processing

If you provide an OpenAI API key and use AI-powered actions:

- selected context data and prompts may be sent to OpenAI APIs
- responses from OpenAI are returned to the Extension and shown in the UI

You are responsible for ensuring this use complies with your institution policies and applicable law.

---

## 5. What We Do Not Intentionally Collect

The Extension is not designed to run a separate user-account backend for telemetry or analytics.

The maintainers do not intentionally collect your personal chat data through a dedicated server in normal extension use.

---

## 6. Your Controls

You can:

- delete all stored sessions from Settings
- remove your saved API key from Settings
- export sessions as JSON

---

## 7. Data Retention

Local data remains until you delete it, uninstall the Extension, or clear browser extension storage.

---

## 8. Security Notice

No software can guarantee absolute security. Keep your browser profile secure and use trusted devices.

---

## 9. Changes to This Policy

This Privacy Policy may be updated. Continued use after updates means you accept the revised policy.

---

## 10. Contact

For privacy questions, contact the project maintainer.
